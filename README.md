# Jarvis

Real-time voice assistant for frontline workers. Cross-team information at conversational latency, with persistent memory, real tool integrations (GitHub, weather), and zero hallucinations.

**Client:** Frontier Audio (Silver tier Gauntlet partner project).
**Spec source:** Gauntlet portal partner project "Jarvis" (last updated 2026-03-16).

## Status

Pre-bootstrap. Five foundational artifacts are populated:

- `constitution.md` — rules, stack constraints, non-negotiables.
- `spec.md` — problem, user stories (US-01..US-08 MVP, US-09..US-12 bonus), acceptance criteria, demo script.
- `plan.md` — architecture, components, decisions, trade-offs, slice sequencing.
- `tasks.md` — checkbox slices the implementing agent picks up.
- `QA_ADVERSARY.md` — vouch sub-agent's project-specific attack surface.

Code lands in `src/` starting Slice 1.
